_Installation mit einem Paket-Manager werden nur in der **x64**-Architektur unterstützt_. Bei anderen Architekturen (z. B. **ARM**) muss das .NET Core SDK oder die .NET Core-Runtime manuell installiert werden. Weitere Informationen finden Sie weiter unten im Abschnitt [Manuelle Installation](#manual-install).
