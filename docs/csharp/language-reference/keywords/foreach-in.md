<span data-ttu-id="4ace9-112">Wenn `SomeType` eine `class` oder `interface` und nicht `sealed` ist, kann der Typ der Elemente außerdem Folgendes enthalten:</span><span class="sxs-lookup"><span data-stu-id="4ace9-112">Furthermore, if `SomeType` is a `class` or an `interface` and not `sealed`, the type of elements may include:</span></span>

- <span data-ttu-id="4ace9-114">eine beliebige Schnittstelle</span><span class="sxs-lookup"><span data-stu-id="4ace9-114">Any arbitrary interface.</span></span> <span data-ttu-id="4ace9-115">Hier ist jede beliebige Schnittstelle zulässig, da jede Schnittstelle von einer Klasse implementiert werden kann, die von `SomeType` abgeleitet oder implementiert wurde.</span><span class="sxs-lookup"><span data-stu-id="4ace9-115">Any interface is allowed because any interface may be implemented by a class derived from or implementing `SomeType`.</span></span>

<span data-ttu-id="4ace9-118">Ab C# 7.3 können Sie die Iterationsvariable mit den Modifizierern `ref` oder `ref readonly` deklarieren, wenn die `Current`-Eigenschaft des Enumerators einen [Verweisrückgabewert](ref.md#reference-return-values) (`ref T`, wobei `T` dem Typ des Auflistungselements entspricht) zurückgibt.</span><span class="sxs-lookup"><span data-stu-id="4ace9-118">Beginning with C# 7.3, if the enumerator's `Current` property returns a [reference return value](ref.md#reference-return-values) (`ref T` where `T` is the type of the collection element), you can declare the iteration variable with the `ref` or `ref readonly` modifier.</span></span>

class="sxs-lookup"><span data-stu-id="4ace9-122">If you want to disable capturing of the context, use the <span data-ttu-id="4ace9-124">Sie können die Schleife an jedem Punkt im `foreach`-Anweisungsblock mit der Anweisung [break](break.md) unterbrechen oder mit der Anweisung [continue](continue.md) direkt zum nächsten Durchlauf der Schleife springen.</span><span class="sxs-lookup"><span data-stu-id="4ace9-124">At any point within the `foreach` statement block, you can break out of the loop by using the [break](break.md) statement, or step to the next iteration in the loop by using the [continue](continue.md) statement.</span></span> <span data-ttu-id="4ace9-125">Sie können eine `foreach`-Schleife auch mit den Anweisungen [goto](goto.md), [return](return.md) oder [throw](throw.md) beenden.</span><span class="sxs-lookup"><span data-stu-id="4ace9-125">You can also exit a `foreach` loop by the [goto](goto.md), [return](return.md), or [throw](throw.md) statements.</span></span>
