# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="86a26-102">Compilerwarnung (Stufe 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="86a26-102">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="86a26-103">"member" ist veraltet</span><span class="sxs-lookup"><span data-stu-id="86a26-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="86a26-104">Der Klassendesigner hat einen Member mit dem [Attribut Obsolete](../language-reference/attributes/general.md#obsolete-attribute)markiert.</span><span class="sxs-lookup"><span data-stu-id="86a26-104">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="86a26-105">Dies bedeutet, dass der Member möglicherweise in einer zukünftigen Version der Klasse unterstützt wird.</span><span class="sxs-lookup"><span data-stu-id="86a26-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="86a26-106">Das folgende Beispiel zeigt, wie der Zugriff auf einen veralteten Member CS0612 generiert:</span><span class="sxs-lookup"><span data-stu-id="86a26-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
