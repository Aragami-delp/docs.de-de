# <a name="exceptionhelper-class"></a>Exceptionhelper-Klasse

Stellt Ausnahmen mit standardisierten Fehlermeldungen bereit. Diese Klasse kann nicht vererbt werden.

```csharp
internal static class ExceptionHelper
```

> [!WARNING]
> Diese Klasse ist intern und sollte nicht direkt im Code verwendet werden.
>
> Microsoft unterstützt die Verwendung dieser Klasse in einer Produktionsanwendung unter keinen Umständen.

## <a name="webpermissionunrestricted-field"></a>Webpermissionuneingeschränktes Feld

Gibt an, dass die APP eine Verbindung mit Internetressourcen herstellen kann.

```csharp
internal static readonly WebPermission WebPermissionUnrestricted
```
