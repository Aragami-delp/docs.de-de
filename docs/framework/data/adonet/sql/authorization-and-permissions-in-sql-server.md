## <a name="ownership-chains"></a><span data-ttu-id="26842-148">Besitzketten</span><span class="sxs-lookup"><span data-stu-id="26842-148">Ownership Chains</span></span>  
 <span data-ttu-id="26842-149">SQL Server stellt sicher, dass nur Prinzipale auf Objekte zugreifen können, denen eine entsprechende Berechtigung gewährt wurde.</span><span class="sxs-lookup"><span
