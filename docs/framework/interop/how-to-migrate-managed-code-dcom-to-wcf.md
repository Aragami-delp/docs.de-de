### <a name="step-1-define-the-sessionful-wcf-service-interface-and-implementation"></a><span data-ttu-id="01a3a-184">Schritt 1: Definieren der sitzungsbasierten WCF-Dienstschnittstelle und -Implementierung</span><span class="sxs-lookup"><span data-stu-id="01a3a-184">Step 1: Define the Sessionful WCF service interface and implementation</span></span>
