# <a name="iapartmentcallbackdocallback-method"></a>IApartmentCallback::DoCallback-Methode
Führt die angegebene Funktion in einem Apartment aus.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT _stdcall DoCallback(  
    [in] SIZE_T pFunc,  
    [in] SIZE_T pData  
);  
```  
  
## <a name="parameters"></a>Parameter  
 `pFunc`  
 in Ein Zeiger auf die Funktion, die im Apartment ausgeführt werden soll.  
  
 `pData`  
 in Ein Zeiger auf das-Argument der Funktion.
