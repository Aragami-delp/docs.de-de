## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags
```  

## <a name="parameters"></a>Parameter  
 `wzFile`

 `dwSaveFlags`  
 [in]: Reserviert Muss Null sein.
