### <a name="deploy-microsoftsparkworker"></a><span data-ttu-id="aacd4-135">Bereitstellen von Microsoft.Spark.Worker</span><span class="sxs-lookup"><span data-stu-id="aacd4-135">Deploy Microsoft.Spark.Worker</span></span>
