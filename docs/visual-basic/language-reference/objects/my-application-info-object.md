# <a name="myapplicationinfo-object"></a>My.Application.Info-Objekt
Stellt Eigenschaften zum Abrufen von Informationen über die Anwendung bereit, z. B. die Versionsnummer, Beschreibung, geladene Assemblys usw.  
  
## <a name="remarks"></a>Bemerkungen  
 Informationen zu den Methoden und Eigenschaften des `My.Application.Info` -Objekts finden Sie unter <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
> Mit den Eigenschaften der-Klasse können Sie <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> Informationen zu einer Datei auf dem Datenträger abrufen.
