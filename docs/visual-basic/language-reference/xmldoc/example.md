# <a name="example-visual-basic"></a>\<example> (Visual Basic)
Gibt ein Beispiel für den Member an.  
  
## <a name="syntax"></a>Syntax  
  
```xml  
<example>description</example>  
```  
  
## <a name="parameters"></a>Parameter  
 `description`  
 Eine Beschreibung des Codebeispiels.  
  
## <a name="remarks"></a>Hinweise  
 Mit dem- `<example>` Tag können Sie ein Beispiel für die Verwendung einer Methode oder eines anderen Bibliotheks Members angeben. Dies umfasst häufig die Verwendung des- [\<code>](code.md) Tags.  
  
 Kompilieren Sie mit [-doc](../../reference/command-line-compiler/doc.md), um Dokumentationskommentare zu einer Datei zu verarbeiten.
