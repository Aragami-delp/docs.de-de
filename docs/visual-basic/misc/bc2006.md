# <a name="option-optionname-requires-argument"></a>Option "\<optionname>" erfordert ""\<argument>

 **Fehler-ID:** BC2006  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Geben Sie das fehlende Argument an.
