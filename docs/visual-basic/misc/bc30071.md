# <a name="case-else-can-only-appear-inside-a-select-case-statement"></a>"Case Else" kann nur innerhalb einer "Select Case"-Anweisung verwendet werden.
Eine `Case Else` -Anweisung befindet sich au√üerhalb eines `Select` -Blocks. Eine `Case Else` -Anweisung kann nur zwischen einer `Select` - oder `Select Case` -Anweisung und der entsprechenden `End Select` -Anweisung verwendet werden.  
  
 **Fehler-ID:** BC30071  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Entfernen Sie die `Case Else` -Anweisung, oder verschieben Sie sie in einen `Select` - Block.
