# <a name="new-cannot-be-used-on-class-classname-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>"New" kann nicht für eine Klasse "\<classname>" verwendet werden, da es einen MustOverride-Member enthält, der nicht überschrieben wurde
Sie haben versucht, `New` für eine Klasse zu verwenden, die einen `MustOverride` -Member enthält, der nicht überschrieben wurde.  
  
 **Fehler-ID:** BC30376  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Entfernen Sie die `New` -Anweisung.
