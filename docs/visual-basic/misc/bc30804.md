# <a name="variant-is-no-longer-a-supported-type-use-the-object-type-instead"></a>Der Typ "Variant" wird nicht mehr unterstützt. Verwenden Sie stattdessen den Typ "Object".
`Object` wird durch den `Variant`-Datentyp ersetzt.  
  
 **Fehler-ID:** BC30804  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Deklarieren Sie das Element als `Object`.
