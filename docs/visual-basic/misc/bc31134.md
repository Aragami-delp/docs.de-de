# <a name="addhandler-and-removehandler-method-parameters-cannot-be-declared-byref"></a>Die Methodenparameter 'AddHandler' und 'RemoveHandler' können nicht 'ByRef' deklariert werden.
Die Parameter der `AddHandler` - und der `RemoveHandler` -Methode können nicht mit dem `ByRef` -Modifizierer deklariert werden.  
  
 **Fehler-ID:** BC31134  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Entfernen Sie das `ByRef` -Schlüsselwort aus der Anweisung.
