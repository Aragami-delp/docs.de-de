# <a name="friend-access-was-denied-to-assembly-if-you-want-the-compiler-to-consider-friend-types-in-assembly-add-the-public-key-information-in-the-project-settings-or-through-a-command-line-switch-to-the-compiler-for-the-project-project"></a>Der friend-Zugriff wurde für \<assembly> verweigert. Wenn der Compiler friend-Typen in "\<assembly>" berücksichtigen soll, fügen Sie für das Projekt "\<project>" die öffentlichen Schlüsselinformationen in den Projekteinstellungen hinzu oder dem Compiler mithilfe eines Befehlszeilenschalters.

**Fehler-ID:** BC31536
