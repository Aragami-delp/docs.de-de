# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Eine Methodendeklarationsanweisung muss die erste Anweisung in einer logischen Zeile sein
Einem Doppelpunkt (:) als Trennzeichen folgt eine `Function` -Anweisung oder `Sub` -Anweisung. `Function` oder `Sub` muss die einzige Anweisung auf der jeweiligen Quellzeile sein.  
  
 **Fehler-ID:** BC32009  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Platzieren Sie jede Anweisung in einer eigenen Zeile.
