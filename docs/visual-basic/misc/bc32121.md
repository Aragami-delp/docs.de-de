# <a name="cannot-inherit-interface-interfacename1-because-the-interface-interfacename2-from-which-it-inherits-could-be-identical-to-interface-interfacename3-for-some-type-arguments"></a><span data-ttu-id="a000d-102">Die Schnittstelle "\<interfacename1>" kann nicht geerbt werden, da die Schnittstelle "\<interfacename2>", von der geerbt wird, bei manchen Typargumenten möglicherweise mit der Schnittstelle "\<interfacename3>" identisch ist</span><span class="sxs-lookup"><span data-stu-id="a000d-102">Cannot inherit interface '\<interfacename1>' because the interface '\<interfacename2>' from which it inherits could be identical to interface '\<interfacename3>' for some type arguments</span></span>
<span data-ttu-id="a000d-103">Eine generische Schnittstelle erbt von mindestens zwei generischen Schnittstellen, und zwei der Vererbungen können einen Konflikt mit bestimmten Werten von Typargumenten verursachen.</span><span class="sxs-lookup"><span data-stu-id="a000d-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="a000d-104">Dieser Fehler kann durch die folgenden Anweisungen generiert werden.</span><span class="sxs-lookup"><span data-stu-id="a000d-104">The following statements can generate this error.</span></span>  
  
```vb  
Public Interface interfaceA(Of u)  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="a000d-105">Wenn die Erstellung oder Implementierung von `derivedInterface` durch Angeben desselben Typs für `t1` und `t2`erfolgt, muss sie zwei Versionen von `interfaceX` mit identischen Typargumenten erben.</span><span class="sxs-lookup"><span data-stu-id="a000d-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="a000d-106">Dies führt zu einer Mehrdeutigkeit in Bezug auf die Version, auf die zugegriffen werden soll.</span><span class="sxs-lookup"><span data-stu-id="a000d-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="a000d-107">**Fehler-ID:** BC32121</span><span class="sxs-lookup"><span data-stu-id="a000d-107">**Error ID:** BC32121</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a000d-108">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="a000d-108">To correct this error</span></span>  
  
- <span data-ttu-id="a000d-109">Ändern Sie eines der für die abgeleitete Schnittstelle angegebenen Typargumente, sodass kein Konflikt auftritt.</span><span class="sxs-lookup"><span data-stu-id="a000d-109">Change one of the type arguments supplied to the derived interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="a000d-110">Oder</span><span class="sxs-lookup"><span data-stu-id="a000d-110">-or-</span></span>  
  
- <span data-ttu-id="a000d-111">Entfernen Sie eine der Schnittstellen, die den potenziellen Vererbungs- oder Implementierungskonflikt auslösen, aus der `Inherits` -Anweisung.</span><span class="sxs-lookup"><span data-stu-id="a000d-111">Remove from the `Inherits` statement one of the interfaces causing the potential inheritance or implementation conflict.</span></span>
