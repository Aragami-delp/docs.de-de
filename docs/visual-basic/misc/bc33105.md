# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="24ab7-102">If-Operanden dürfen keine benannten Argumente sein</span><span class="sxs-lookup"><span data-stu-id="24ab7-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="24ab7-103">Die Verwendung von benannten Argumenten in den Operanden des `If` -Operators ist ungültig.</span><span class="sxs-lookup"><span data-stu-id="24ab7-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="24ab7-104">Dieser Fehler wird beispielsweise durch den folgenden Code verursacht.</span><span class="sxs-lookup"><span data-stu-id="24ab7-104">The following example causes this error:</span></span>  
  
```vb  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="24ab7-105">Dies stellt einen Unterschied zur `IIf` -Funktion dar, die benannte Argumente zulässt, wie im folgenden Code zu sehen:</span><span class="sxs-lookup"><span data-stu-id="24ab7-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="24ab7-106">**Fehler-ID:** BC33105</span><span class="sxs-lookup"><span data-stu-id="24ab7-106">**Error ID:** BC33105</span></span>
