# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="cf0ed-102">Es kann kein gemeinsamer Typ für den ersten und zweiten Operanden des binären If-Operators abgeleitet werden.</span><span class="sxs-lookup"><span data-stu-id="cf0ed-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="cf0ed-103">Es kann kein gemeinsamer Typ für den ersten und zweiten Operanden des binären If-Operators abgeleitet werden.</span><span class="sxs-lookup"><span data-stu-id="cf0ed-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="cf0ed-104">Für einen muss eine erweiternde Konvertierung in den anderen durchgeführt werden.</span><span class="sxs-lookup"><span data-stu-id="cf0ed-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="cf0ed-105">Der binäre `If` -Operator erfordert, dass zwischen einem der Argumente und dem anderen Argument eine erweiternde Konvertierung erfolgt.</span><span class="sxs-lookup"><span data-stu-id="cf0ed-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="cf0ed-106">Da z. B. zwischen `Integer` und `String`in beide Richtungen keine erweiternde Konvertierung durchgeführt wird, verursacht folgender Code diesen Fehler.</span><span class="sxs-lookup"><span data-stu-id="cf0ed-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="cf0ed-107">**Fehler-ID:** BC33110</span><span class="sxs-lookup"><span data-stu-id="cf0ed-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="cf0ed-108">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="cf0ed-108">To correct this error</span></span>

- <span data-ttu-id="cf0ed-109">Stellen Sie eine explizite Konvertierung für einen der Operanden bereit, wenn dies in Ihrem Code möglich ist:</span><span class="sxs-lookup"><span data-stu-id="cf0ed-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="cf0ed-110">Verwenden eine andere bedingte Konstruktion, um den Code neu zu schreiben.</span><span class="sxs-lookup"><span data-stu-id="cf0ed-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
