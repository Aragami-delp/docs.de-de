## <a name="example"></a><span data-ttu-id="8dcd9-111">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8dcd9-111">Example</span></span>

<span data-ttu-id="8dcd9-112">Im folgenden Beispiel wird das `AttributeUsage` -Attribut verwendet, um anzugeben, auf welche Programmelemente die neue Version von `ExtensionAttribute` angewendet werden kann.</span><span class="sxs-lookup"><span data-stu-id="8dcd9-112">The following example uses the `AttributeUsage` attribute to specify which program elements the new version of `ExtensionAttribute` can apply to.</span></span> <span data-ttu-id="8dcd9-113">Das Beispiel gibt drei Member der `AttributeTargets` -Enumeration an: `Assembly`, `Class`und `Method`.</span><span class="sxs-lookup"><span data-stu-id="8dcd9-113">The example specifies three members of the `AttributeTargets` enumeration: `Assembly`, `Class`, and `Method`.</span></span> <span data-ttu-id="8dcd9-114">Die Auslassung eines dieser Elemente verursacht diesen Fehler.</span><span class="sxs-lookup"><span data-stu-id="8dcd9-114">The omission of any one of these elements will cause this error.</span></span>

```vb
Namespace System.Runtime.CompilerServices
    <AttributeUsage(AttributeTargets.Assembly Or _
        AttributeTargets.Class Or AttributeTargets.Method)>
    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```

<span data-ttu-id="8dcd9-115">Alternativ können Sie `ExtensionAttribute` die Anwendung auf alle Programmelemente mithilfe des `All` -Members von `AttributeTargets`gestatten.</span><span class="sxs-lookup"><span data-stu-id="8dcd9-115">Alternatively, you could allow `ExtensionAttribute` to apply to all program elements by using the `All` member of `AttributeTargets`.</span></span>

```xml
<AttributeUsage(AttributeTargets.All)>
```

<span data-ttu-id="8dcd9-116">Das Löschen der `AttributeUsage` -Zeile führt zum gleichen Ergebnis, wie im folgenden Code gezeigt.</span><span class="sxs-lookup"><span data-stu-id="8dcd9-116">Deleting the `AttributeUsage` line, as shown in the following code, produces the same result.</span></span>

```vb

    Class ExtensionAttribute
        Inherits System.Attribute
        ' Definitions of methods, fields, and properties.
    End Class
End Namespace
```
