# <a name="instance-of-restricted-type-typename-cannot-be-used-in-a-lambda-expression"></a>Eine Instanz des eingeschränkten Typs "\<typename>" darf nicht in einem lambda-Ausdruck verwendet werden
Instanzen von eingeschränkten Typen, wie z. B. <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>und <xref:System.TypedReference>dürfen nicht in einem Lambda- oder Abfrageausdruck verwendet werden.  
  
 **Fehler-ID:** BC36640
