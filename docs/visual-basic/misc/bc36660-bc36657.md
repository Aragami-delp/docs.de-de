# <a name="data-types-of-the-type-parameters-in-method-methodname-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="e0b93-102">Die Datentypen der Typparameter in der \<methodname>-Methode können nicht von diesen Argumenten abgeleitet werden, da sie nicht in denselben Typ konvertiert werden</span><span class="sxs-lookup"><span data-stu-id="e0b93-102">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type</span></span>

<span data-ttu-id="e0b93-103">Die Datentypen der Typparameter in der-Methode \<methodname> können nicht von diesen Argumenten abgeleitet werden, da Sie nicht in denselben Typ konvertiert werden.</span><span class="sxs-lookup"><span data-stu-id="e0b93-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="e0b93-104">Sie können diesen Fehler möglicherweise beheben, indem Sie die Datentypen explizit angeben.</span><span class="sxs-lookup"><span data-stu-id="e0b93-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="e0b93-105">Beim Auswerten eines Aufrufs einer generischen Prozedur wurde versucht, über den Typrückschluss den Datentyp (bzw. die Datentypen) der Typparameter zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="e0b93-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="e0b93-106">Der Compiler konnte keinen Datentyp finden, der die Einschränkungen aller Argumente erfüllt.</span><span class="sxs-lookup"><span data-stu-id="e0b93-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="e0b93-107">Daher wurde dieser Fehler gemeldet.</span><span class="sxs-lookup"><span data-stu-id="e0b93-107">Therefore, it reported this error.</span></span>

> [!NOTE]
> <span data-ttu-id="e0b93-108">Wenn die Angabe von Argumenten keine Option ist (z. B. für Abfrageoperatoren in Abfrageausdrücken), wird nur der erste Satz der Fehlermeldung angezeigt.</span><span class="sxs-lookup"><span data-stu-id="e0b93-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="e0b93-109">Im folgenden Code wird der Fehler veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="e0b93-109">The following code demonstrates the error.</span></span>

```vb

Module Module1
    Sub Main()

        '' Not valid. Integer and Date do not convert to the same type.
        'targetMethod(19, #3/4/2007#)

    End Sub

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)
    End Sub

End Module
```

<span data-ttu-id="e0b93-110">**Fehler-ID:** BC36660 und BC36657</span><span class="sxs-lookup"><span data-stu-id="e0b93-110">**Error ID:** BC36660 and BC36657</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="e0b93-111">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="e0b93-111">To correct this error</span></span>

