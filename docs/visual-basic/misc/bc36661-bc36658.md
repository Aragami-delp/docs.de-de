<span data-ttu-id="cddf0-103">Die Datentypen der Typparameter in der \<methodname> in "Typname" definierten Erweiterungsmethode "" können nicht von diesen Argumenten abgeleitet werden, da Sie nicht in denselben Typ konvertiert werden.</span><span class="sxs-lookup"><span data-stu-id="cddf0-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="cddf0-104">Sie können diesen Fehler möglicherweise beheben, indem Sie die Datentypen explizit angeben.</span><span class="sxs-lookup"><span data-stu-id="cddf0-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="cddf0-105">Beim Auswerten eines Aufrufs einer generischen Erweiterungsmethode wurde versucht, über den Typrückschluss den Datentyp (oder die Datentypen) des Typparameters (oder der Typparameter) zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="cddf0-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="cddf0-106">Der Compiler konnte keinen Datentyp finden, der die Einschränkungen aller Argumente erfüllt.</span><span class="sxs-lookup"><span data-stu-id="cddf0-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="cddf0-107">Daher wurde dieser Fehler gemeldet.</span><span class="sxs-lookup"><span data-stu-id="cddf0-107">Therefore, it reported this error.</span></span>
