# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>Die Länge des Arguments \<argumentname> muss größer als 0 (null) sein
Ein Wert eines Arguments ist gleich oder kleiner als 0 (null).  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Geben Sie ein Argument an, das größer als 0 (null) ist.
