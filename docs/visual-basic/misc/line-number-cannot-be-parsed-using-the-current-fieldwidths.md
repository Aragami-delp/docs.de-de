# <a name="line-number-cannot-be-parsed-using-the-current-fieldwidths"></a>Die Zeile \<number> kann mit den aktuellen FieldWidths nicht analysiert werden
Die angegebene Zeile kann nicht analysiert werden, da ihre Felder nicht die angegebenen Breiten haben.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Passen Sie `FieldWidths` so an, dass die Zeile fehlerfrei analysiert werden kann, oder f√ºgen Sie Ausnahmebehandlungscode ein, um die Zeile zu verarbeiten.
