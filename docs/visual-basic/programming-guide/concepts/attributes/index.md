## <a name="using-attributes"></a><span data-ttu-id="759dc-116">Verwenden von Attributen</span><span class="sxs-lookup"><span data-stu-id="759dc-116">Using Attributes</span></span>

<span data-ttu-id="759dc-117">Attribute können in nahezu jeder Deklaration platziert werden, auch wenn ein bestimmtes Attribut die Typen der Deklarationen einschränkt, für die es gültig ist.</span><span class="sxs-lookup"><span data-stu-id="759dc-117">Attributes can be placed on most any declaration, though a specific attribute might restrict the types of declarations on which it is valid.</span></span> <span data-ttu-id="759dc-118">In Visual Basic wird ein Attribut in eckige Klammern () eingeschlossen \< > .</span><span class="sxs-lookup"><span data-stu-id="759dc-118">In Visual Basic, an attribute is enclosed in angle brackets (\< >).</span></span> <span data-ttu-id="759dc-119">Es muss in derselben Zeile direkt vor dem Element angezeigt werden, auf das es angewendet wird.</span><span class="sxs-lookup"><span data-stu-id="759dc-119">It must appear immediately before the element to which it is applied, on the same line.</span></span>


 <span data-ttu-id="759dc-121">Eine Methode mit dem Attribut <xref:System.Runtime.InteropServices.DllImportAttribute> wird wie folgt deklariert:</span><span class="sxs-lookup"><span data-stu-id="759dc-121">A method with the attribute <xref:System.Runtime.InteropServices.DllImportAttribute> is declared like this:</span></span>



<span data-ttu-id="759dc-122">Mehrere Attribute können in einer Deklaration platziert werden:</span><span class="sxs-lookup"><span data-stu-id="759dc-122">More than one attribute can be placed on a declaration:</span></span>
