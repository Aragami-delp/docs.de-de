### <a name="compiler-support-for-type-forwarding-when-multi-targeting-mscorlib"></a>Compilerunterstützung für Typweiterleitung bei Festlegung von Zielversionen für das mscorlib-Element

#### <a name="details"></a>Details

Durch ein neues CodeDOM-Feature kann ein Compiler anhand der Zielversion von mscorlib.dll anstelle der .NET Framework 4.5-Version von mscorlib.dll kompilieren.

| name    | Wert       |
|:--------|:------------|
| Bereich   |Microsoft Edge|
|Version|4.5|
|Typ|Laufzeit|
